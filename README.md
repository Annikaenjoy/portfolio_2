## Getting Started

### Installing

How to get started, step by step:

1. Clone the repo:

```bash
git https://github.com/Annikaenjoy/semester_project2.git
```

2. Install the dependencies:

```
npm install
```

### Running

To run the app, run the following commands:

```bash
npm run start
```
